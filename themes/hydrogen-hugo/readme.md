## Local development

```bash
# clone the repository
git clone git@github.com:statichunt/hydrogen-hugo.git

# cd in the project directory
$ npm install

# cd in the project directory
$ cd hydrogen-hugo/exampleSite/

# Start local dev server
$ hugo server --themesDir ../..
```
